# ODONTO-SCAN — Dental Biometric Authentication System

A biometric-based authentication system for secure dental and medical record management, developed as a Final Year Project at University of Agriculture Faisalabad.

## What it does
Identifies patients using dental biometric patterns for secure medical record access, reducing manual verification errors and improving clinical workflow efficiency.

## Technologies Used
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Computer Vision:** OpenCV
- **Security:** Encryption techniques for data privacy

## Features
- Biometric dental pattern recognition
- Encrypted patient record storage
- Secure authentication system
- Clinical workflow optimization
